## 웹 서비스 기획서: [당신의 독자]

### 1. 프로젝트 개요

- **프로젝트명:** 당신의 독자 - 마음을 읽어주는 가상의 에세이 서재
- **기획 목적:** 평가나 교정이 아닌, 오롯이 '공감'에 초점을 맞춘 글쓰기 경험 제공. 글쓴이의 감정을 온전히 수용하고 따뜻한 감상을 건네는 감성 웹 서비스 구축.
- **타겟 유저:**
    - 일기나 에세이를 쓰며 스스로의 감정을 정리하고 구체화 하고 싶은 사람

### 2. 핵심 서비스 경험 (User Flow)

1. **접속 및 페르소나 선택:** 사용자가 웹사이트에 접속해 오늘 자신의 글을 읽어줄 '가상의 독자(페르소나)'를 선택합니다.
2. **글쓰기 몰입 (에디터):** 백색소음과 여백이 많은 차분한 UI 속에서 에세이를 작성합니다.
3. **감정 분석 및 피드백:** 글 작성을 완료하면 허깅페이스 AI 모델이 글의 감정을 분석하고, 선택한 독자의 말투로 따뜻한 감상평(답글)을 남겨줍니다.
4. **공감의 시각화:** 글 속에서 독자의 마음을 가장 크게 울린 특정 문장에 따뜻한 색상의 형광펜 하이라이트가 칠해집니다.
5. **기록 보관:** 작성한 글과 독자의 답글, 분석된 감정 통계가 개인 보관함에 저장됩니다.
6. **추가 확장**:
    - **글쓰기 몰입 및 페르소나 선택:** 사용자가 '한국어에 서툴지만 마음이 따뜻한 외국인 펜팔 친구' 페르소나를 선택합니다.
    - **번역 및 글로벌 공감 (추가):** 한글로 에세이를 작성하면, 외국인 독자가 글의 감정을 분석한 뒤 **① 영어로 작성된 따뜻한 답글**과 **② 사용자의 글을 자연스러운 감성 영문으로 번역한 완성본**을 함께 전달합니다.

### 3. 주요 기능 명세

| **기능명** | **상세 내용** |
| --- | --- |
| **몰입형 에디터** | 글쓰기에 온전히 집중할 수 있는 미니멀 UI 및 자동 저장 기능 |
| **독자 페르소나 설정** | 따뜻한 위로형, 차분한 경청형 등 다채로운 가상 독자 프로필 제공 |
| **AI 감정 분석** | 허깅페이스 오픈소스 모델을 활용해 글의 지배적인 감정 수치화 |
| **공감 코멘트 생성** | 분석된 감정 데이터에 기반하여 페르소나 성격에 맞는 감상평 생성 |
| **감동 문장 하이라이팅** | 문법 오류 지적이 아닌, 감정선이 가장 짙은 문장을 시각적으로 강조 |
| **나만의 서재 (대시보드)** | 과거 작성 글 조회 및 누적된 감정 데이터를 활용한 월간 리포트 제공 |
| **외국인 독자 페르소나** | 한국어 텍스트를 읽고 영어로 공감 및 번역을 제공하는 다국어 펜팔 컨셉의 가상 독자 |
| **감성 영문 번역** | 단순 직역이 아닌, 사용자의 감정선과 문맥을 살려 자연스러운 영문 에세이로 재창조  |

### 4. 시스템 구조 및 기술 스택

| **구분** | **도입 기술 (예정)** | **활용 목적** |
| --- | --- | --- |
| **Frontend** | React / Next.js | 인터랙티브한 화면 구성 및 웹 텍스트 에디터 연동 |
| **Backend & DB** | Supabase | 회원가입/로그인 처리 및 작성된 에세이, 피드백 결과의 안전한 저장 |
| **Deployment** | Vercel | 프론트엔드 무료 호스팅 및 빠르고 간편한 배포 파이프라인 구축 |
| **AI (감정 분석)** | Hugging Face (KoBERT / KcBERT 기반) | 텍스트 내 감정 다중 분류 (슬픔, 위로, 기쁨 등을 수치화하여 데이터로 저장) |
| **AI (문장 생성)** | Hugging Face (오픈소스 한국어 생성 모델) | 수치화된 감정 결과를 바탕으로 자연스러운 공감의 답글(Text) 생성 |
| **AI(기계번역)** | Hugging Face (NLLB 등 오픈 소스 번역 모델) | 한글 기반 에세이를 감성적인 영어 문장으로 번역 및 영어 답글 생성 |

### 5. 프로젝트 차별점 및 기대 효과

- **평가가 아닌 공감:** 기존의 '글쓰기 튜터' 도구들을 벗어나, 심리적 안정과 위로를 주는 '감성 테크(Emotion Tech)' 포지셔닝으로 독창적인 시장 개척.
- **비용 효율성 극대화:** Vercel(무료 호스팅), Supabase(무료 DB), Hugging Face(무료 오픈소스 AI)를 결합하여 초기 서버 유지 및 AI 추론 비용을 0원에 가깝게 설계.
- **데이터 자산화:** 감정 분석 결과가 축적되면 사용자에게 시각화된 개인화 리포트를 제공하여, 서비스 재방문율과 애착도를 크게 높일 수 있음.
- **자연스러운 어학 학습 연계:** 자신의 이야기가 담긴 글을 영어로 확인하게 되므로, 일반적인 번역기나 예문보다 훨씬 높은 몰입도와 언어 학습 효과를 제공합니다.
- **글로벌 서비스로의 확장 가능성:** '외국인 펜팔' 컨셉이 성공적으로 안착하면, 향후 반대로 영어나 다른 언어로 작성된 글을 한국어로 번역하고 공감해 주는 역방향 서비스로도 쉽게 확장할 수 있습니다.
